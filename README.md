git init → for initialising the local repo
git status → view the status of the initialised repo
git add <file> → to add a file to staging area
Or 
git add . → to add all files to the staging area
git commit -m “commit message”

To set up the configuration:
git config --(local/global) user.name “Name”
git config --(local/global) user.email “email”

git branch —--- to view the git branch
